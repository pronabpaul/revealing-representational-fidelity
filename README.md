# Beyond Reconstruction: Evaluating Representational Fidelity in Continuous Neural Representations

Implementation accompanying the technical report **"Beyond Reconstruction: Evaluating Representational Fidelity in Continuous Neural Representations."**

## Overview

Continuous Neural Representations (CNRs) are typically evaluated using reconstruction metrics such as Dice, HD95, and ASD. This work demonstrates that **equivalent reconstruction performance does not necessarily imply equivalent learned representations**.

The report introduces the **Representational Fidelity Index (RFI)**, a representation-aware evaluation framework that combines geometric fidelity and latent representation richness to quantify how faithfully anatomical information is preserved within learned representations. The framework is validated on the **DRIVE** and **GLAS** datasets.
